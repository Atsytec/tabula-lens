# React API Reference

Complete API reference for the `@tabula-lens/react` package.

## DatabaseViewer Component

The main component for displaying database data in your React application.

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `endpoint` | `string` | Yes | The API endpoint to query for data |
| `authToken` | `string` | No | Authentication token for API requests |
| `getAuthHeaders` | `function` | No | Function to return auth headers |
| `styles` | `object` | No | Custom styles for component |
| `classNames` | `object` | No | Custom class names for component |
| `pageSize` | `number` | No | Number of rows per page (default: 50) |
| `tableSelector` | `'dropdown' \| 'sidebar'` | No | Table selector mode (default: 'dropdown') |
| `filterPosition` | `'top' \| 'bottom'` | No | Filter input position (default: 'top') |
| `paginationPosition` | `'top' \| 'bottom'` | No | Pagination position (default: 'bottom') |

### Example

```jsx
<DatabaseViewer 
  endpoint="/api/tabula-lens"
  authToken={userToken}
  pageSize={25}
  tableSelector="sidebar"
/>
```

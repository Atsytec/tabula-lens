Tabula Lens consists of two parts: a backend SDK for Node.js and a React component for the frontend. Here's how to get started.

## Backend Setup

First, install the backend SDK:

```bash
npm install @tabula-lens/node
```

Then, create an API endpoint in your backend:

```javascript
import TabulaLens from "@tabula-lens/node"

const tabulaLens = new TabulaLens(process.env.DATABASE_URL)

// Example with Express
app.get('/api/tabula-lens', async (req, res) => {
  const data = await tabulaLens.query(req.query)
  res.json(data)
})
```

## Frontend Setup

Install the React component:

```bash
npm install @tabula-lens/react
```

Use the component in your React application:

```jsx
import { DatabaseViewer } from '@tabula-lens/react'

function App() {
  return (
    <DatabaseViewer 
      endpoint="/api/tabula-lens"
      authToken={userToken}
    />
  )
}
```

## Requirements

- **Backend**: Node.js 18+, PostgreSQL database
- **Frontend**: React 18+

# Accessibility Statement

Tabula Lens is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

## Our Commitment

We strive to ensure that our documentation website is accessible to people with disabilities, including:

- Visual impairments (blindness, low vision, color blindness)
- Motor impairments (limited mobility, tremors)
- Cognitive impairments (attention deficits, learning disabilities)
- Hearing impairments

## Accessibility Standards

Our documentation aims to comply with:

- **WCAG 2.1 Level AA** - Web Content Accessibility Guidelines
- **Section 508** - U.S. federal accessibility standards
- **EN 301 549** - European accessibility requirements

## Accessibility Features

### Visual Accessibility

**✓ High Contrast Mode**
- Supports system high contrast preferences
- Enhanced color contrast ratios (minimum 4.5:1 for normal text, 3:1 for large text)
- Clear visual focus indicators for keyboard navigation

**✓ Color Independence**
- Information is not conveyed by color alone
- Text and icons maintain contrast in both light and dark modes
- Color combinations meet WCAG AA requirements

**✓ Responsive Design**
- Content reflows properly on different screen sizes
- Text can be zoomed up to 200% without loss of content
- Touch targets are at least 44x44 pixels for mobile users

### Motor Accessibility

**✓ Keyboard Navigation**
- All interactive elements are keyboard accessible
- Logical tab order throughout the site
- Visible focus indicators on all interactive elements
- Skip-to-content link for keyboard users

**✓ Reduced Motion**
- Respects `prefers-reduced-motion` system preference
- Optional to disable animations and transitions
- No content relies on animation to convey meaning

### Cognitive Accessibility

**✓ Clear Language**
- Plain language explanations
- Consistent terminology throughout
- Clear headings and structure
- Technical terms are defined when first used

**✓ Consistent Navigation**
- Predictable layout and navigation
- Clear page titles and headings
- Breadcrumb navigation for context
- Search functionality for finding content

### Assistive Technology Support

**✓ Screen Reader Support**
- Semantic HTML structure
- ARIA labels and roles where needed
- Alt text for images
- Proper heading hierarchy
- Screen reader-only content for context

**✓ Text-to-Speech**
- Content is structured for TTS compatibility
- Mathematical notation is properly formatted
- Code blocks are readable by screen readers

## Known Limitations

While we strive for full accessibility, some limitations may exist:

### Third-Party Content

- Some code examples may reference external libraries with varying accessibility
- External links may lead to content not under our control

### Technical Documentation

- Complex code examples may be challenging for some screen reader configurations
- Mathematical notation and technical symbols may require specialized tools

## Accessibility Testing

We regularly test our documentation using:

- Automated accessibility testing tools
- Keyboard-only navigation testing
- Screen reader testing (NVDA, JAWS, VoiceOver)
- Color contrast validation
- Mobile device testing with screen readers

## Feedback and Contact

We welcome feedback on the accessibility of our documentation. Please let us know if you encounter accessibility barriers:

### Reporting Accessibility Issues

If you find an accessibility barrier, please:

1. Describe the specific accessibility issue
2. Provide the URL of the page where you encountered the issue
3. Specify the browser, screen reader, or assistive technology you were using
4. Share any suggestions for improvement

### Contact Methods

- **GitHub Issues**: [Report an issue on GitHub](https://github.com/Atsytec/tabula-lens/issues)

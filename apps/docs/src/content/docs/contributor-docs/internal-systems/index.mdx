# Internal Systems

This section covers internal systems and infrastructure used by Tabula Lens.

## Topics

- **[Design System](./design-system/)** - Design tokens, CSS custom properties, and theming
- **[Error Handling](./error-handling/)** - Error handling patterns and error catalog

## Design System

The Tabula Lens design system includes:

- **Color Palette** - Blue-teal primary colors with warm accents
- **Typography** - Clean, modern sans-serif fonts
- **Spacing** - Consistent spacing scale
- **Border Radius** - Rounded corners for modern UI
- **CSS Custom Properties** - Easy theming and customization
- **Dark Mode** - Built-in dark mode support

## Error Handling

Error handling in Tabula Lens includes:

- **Structured Errors** - Consistent error format across all components
- **Error Catalog** - Comprehensive catalog of error types and solutions
- **User-Friendly Messages** - Clear error messages for end users
- **Developer-Friendly Debugging** - Detailed error information for debugging

## Contributing

If you're interested in contributing to Tabula Lens, please review the contributing guidelines in the main repository. Key principles:

- **Docs-as-Code** - No feature is complete without documentation
- **Test Coverage** - All changes must include tests
- **Type Safety** - Maintain strict TypeScript typing
- **Performance** - Consider performance implications of changes
- **Security** - Follow security best practices

## Architecture Overview

Tabula Lens follows a modular architecture with clear separation of concerns:

- **HTTP API** - Universal interface between frontend and backend
- **Frontend** - React components with modular sub-components
- **Backend** - Node.js server with framework adapters
- **Database** - PostgreSQL with connection pooling

This architecture enables interchangeability of implementations while maintaining a consistent interface.

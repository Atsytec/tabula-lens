# Frontend Guides

These guides cover integrating Tabula Lens into frontend applications, with a focus on React implementations.

## Guides

- **[Frontend Implementation](./frontend-implementation/)** - Complete guide to integrating Tabula Lens into your frontend application
- **[TanStack Query Integration](./tanstack-query-integration/)** - Using TanStack Query for data fetching and caching
- **[TanStack Table Integration](./tanstack-table-integration/)** - Advanced table features with TanStack Table
- **[Styling Customization](./styling-customization/)** - Customizing the appearance of Tabula Lens components

## Key Concepts

The Tabula Lens frontend is built around:

- **HTTP API Contract** - All frontend components communicate with the backend via a well-defined HTTP API
- **Modular Components** - The DatabaseViewer component is composed of smaller, reusable sub-components
- **Custom Hooks** - Reusable hooks for state management, data fetching, and logging
- **Styling System** - CSS custom properties for easy theming and customization

## React Implementation

The primary frontend implementation is for React, but the HTTP API is framework-agnostic. You can use any frontend framework that can make HTTP requests to the Tabula Lens backend.

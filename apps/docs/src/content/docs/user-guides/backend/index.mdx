# Backend Guides

These guides cover setting up and configuring the Tabula Lens backend server and database integration.

## Guides

- **[Backend Implementation](./backend-implementation/)** - Complete guide to setting up the Tabula Lens backend server
- **[Database Integration](./database-integration/)** - Configuring PostgreSQL database connections
- **[Logging System](./logging-system/)** - Understanding and configuring the logging system

## Key Concepts

The Tabula Lens backend is built around:

- **HTTP API** - RESTful API that serves database queries and results
- **Framework Adapters** - Support for 15+ Node.js frameworks (Express, Fastify, Hono, etc.)
- **Database Connection** - PostgreSQL connection pooling and query execution
- **Security** - Built-in authentication and authorization support
- **Logging** - Structured logging with request tracing

## Framework Support

The Tabula Lens backend supports multiple Node.js frameworks:

- Express
- Fastify
- Hono
- Koa
- NestJS
- And 10+ more

See the [Backend Implementation](./backend-implementation/) guide for the complete list and setup instructions.

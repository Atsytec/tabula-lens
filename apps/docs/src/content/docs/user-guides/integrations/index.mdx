# Integration Guides

These guides cover integrating Tabula Lens with authentication providers, hosted databases, ORMs, and developer tooling.

- **[Auth.js](./authjs)** — NextAuth v5 for Next.js and Express
- **[Auth0](./auth0)** — Auth0 JWT validation
- **[Better Auth](./better-auth)** — Framework-agnostic TypeScript auth

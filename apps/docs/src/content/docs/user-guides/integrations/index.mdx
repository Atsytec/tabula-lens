# Integration Guides

These guides cover integrating Tabula Lens with other systems, services, and authentication providers.

## Guides

- **[Authentication](./authentication/)** - Implementing authentication and authorization with Tabula Lens

## Authentication Patterns

Tabula Lens supports multiple authentication patterns:

- **JWT Authentication** - Token-based authentication with JSON Web Tokens
- **Session-based Authentication** - Traditional session management
- **OAuth 2.0** - Integration with OAuth providers (Google, GitHub, etc.)
- **Role-based Access Control (RBAC)** - Fine-grained permissions
- **Row-level Security** - Database-level row access control

## Integration Examples

The authentication guide includes examples for:

- Next.js authentication
- Express.js authentication
- Custom authentication implementations
- Multi-tenant authentication strategies

If you're new to Tabula Lens, start with the [Quick Start](../quick-start/getting-started/) guide to get up and running quickly. Then explore the guides relevant to your use case.

## Architecture Overview

Tabula Lens is built around a modular architecture with clear separation between frontend and backend:

- **HTTP API** - The universal interface that connects frontend and backend implementations
- **Frontend** - React components that consume the HTTP API
- **Backend** - Node.js server that exposes the HTTP API
- **Database** - PostgreSQL database that stores your data

This architecture allows you to mix and match implementations based on your needs.
